
Topic: Password Generator

Task:Create a password generator

Requirements:
- The password must contain 4-7 characters
- The password must contain atleast one capitalized character
- The password must contain atleast 2-3 numbers

Features Breakdown:

Starting: create an empty string to contain all characters before being switched index within itself

1) Create a Math function to randomly generate 4-7 characters and store it in a variable
2) Create a math function to randomly select atleast one character to capitalize
3) Create a math function to randomly generate 2-3 numbers and store it in a variable
4) Join all variables
5) Switch all characters
6) Return the finalized password in string
